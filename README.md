## EsMsg V1.0 -简约弹窗层插件
- 基于jquery开发的简约弹窗层插件(故需要提前引入jquery可使用)，适用于PC端以及移动端，调用简单使用方便。

## 目录层级
- esmsg  [插件包]
    - img [插件图片]
        - ...
    - lib [外部包]
        - jquery
        - ...
    - esmsg.css  [插件样式]
    - esmsg.js   [插件逻辑]
- test   [调用事例]
    - index.html [插件调用事例]

## 使用方案

    简约提示 [esmsg.msg]
    ```
    esmsg.msg("提示信息");
    ```
